<h3>Options:</h3>

<i>For Windows Vista, XP, 7, 10, 11 OS & [WSL](https://learn.microsoft.com/en-us/windows/wsl/basic-commands) users: </i>

Insert the global PATH:

``` 
CMD
cd %AppData%\Code\User && find /c "{" keybindings.json && %AppData%\Code\User\keybindings.json 
```

1. With a <b>File Explorer</b>.
2. Using <b>[`CMD Here`](https://github.com/EstebanMqz/Registries/tree/main)</b> <i>Context Menu Background Registry</i> in any <b>File Explorer</b>.
